## Project Idea

Btracker is a simple task management app where users can add, update, complete, and delete their tasks. It helps users keep track of their daily tasks and stay organized.

## Implemented Features

* User authentication (login and register)
* Add new tasks with title and description
* View all pending tasks on **My Tasks** page
* Mark tasks as completed on **Completed Tasks** page
* Update or delete tasks
* Search tasks by title
* Responsive design for mobile and desktop

## Tech Stack

* **Frontend:** React, TailwindCSS
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Authentication:** Firebase Auth
* **Other:** SweetAlert2 for notifications, React Router for routing

## Challenges & Future Improvements

* **Challenges:** Handling state updates after task changes, filtering tasks by status, and making the UI responsive.
* **Future Improvements:**

  * Add task due dates and reminders
  * Allow task categories or priorities
  * Add drag-and-drop to reorder tasks
  * Improve UI/UX with animations and better design
  * Implement dark mode
